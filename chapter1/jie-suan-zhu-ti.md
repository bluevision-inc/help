#### 常见问题：结算主体

---

##### **为什么要认证结算主体？** {#payentity}

> ```
> 结算主体是与蓝瀚互动进行结算的公司主体，在向广告账户里充值时，结算主体需要在蓝瀚互动系统中认证通过。
> ```

**结算主体不认证通过可以开通广告账户吗？**

> ```
> 不可以
>
> 广告账户在结算主体认证通过钱无法通过审不可以
> ```

**结算主体可以是国内公司吗？**

> ```
> 暂时不可以
>
> 现在只支持美金结算，结算主体需为海外公司主体
> ```

**如何在线上进行合同的签署？**

> ```
> 1.下载合同，并打印
>
> 2.完善以下信息
>   注意：合同编号不需要填写
>     第1页：“甲方名称” “地址” 
>     第3页： “甲方账户信息”
>     第10页：“甲方联系人信息”
>     第11页：“盖章”“合同签订日期”
>     
> 3.将11页合同进行拍照，上传提交即可
> ```
